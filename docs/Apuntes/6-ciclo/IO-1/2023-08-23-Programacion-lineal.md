Los modelos de PL son de muchos tamaños diferentes. Hay casos en los que un modelo tiene 12 o 17 variables incluso, United Airlines tuvo un modelo con 20,000 variables.
Es común que un modelo tenga cientos o miles de restricciones funcionales. Formular un modelo tan grande puede ser una tarea desalentadora.

# Lenguajes de modelos

Un lenguaje de modelo de programación matemática es un software de diseño especial para formular un modelo eficiente. Se han desarrollado varios lenguajes de modelado excelentes en las últimas décadas, entre ellas AMPL, MPL, GAMS, LINGO.
Usar un lenguaje de modelo agiliza el proceso.
